### Creation Setup

- create `manifest.json`
- create icons for the manifest
- `npm init -y`

```bash
npm i -D \
   webpack \
   webpack-cli \
   webpack-dev-server
```

```bash
npm i -D \
   @babel/core \
   @babel/plugin-proposal-class-properties \
   @babel/preset-env \
   @babel/preset-react \
   babel-loader \
   copy-webpack-plugin \
   clean-webpack-plugin \
   css-loader \
   html-webpack-plugin
```
